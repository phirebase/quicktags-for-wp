== Description ==

This simply plugin adds usefull buttons to native WordPress HTML editor. 

Currently adds:
h1, h2, h3, h4, h5, hr, nbsp, mox

Mox - this button adds nice responsive CSS class wich you can use to highlight picture or other elements in your posts.  

== Installation ==

1. Unzip into your `/wp-content/plugins/` directory. If you're uploading it make sure to upload the top-level folder. Don't just upload all the php files and put them in `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress
